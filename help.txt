Горячие клавиши для работы с фалом:
Ctrl+Z — Отмена
Ctrl+Y — Возврат
Ctrl+S — Сохранить файл
Ctrl+Shift+Z -- Создать новый файл

Горячие клавиши для перемещения объекта:
Воспользуйтесь соответствующей клавишей на клавиатуре для
← — Перемещения объекта влево
→ — Перемещения объекта вправо
↑ — Перемещения объекта вверх
↓ — Перемещения объекта вниз

Сочетание клавиш для изменения размера объекта:
Ctrl+P — Увеличение размера
Ctrl+M — Уменьшение размера

Про перемещение слоев:
При нажатии на кнопку ⬇️ самый верхний (последний) элемент
перемещается в самый низ

При нажатии на кнопку ⬆️ самый нижний (первый) элемент
перемещается в самый верх
